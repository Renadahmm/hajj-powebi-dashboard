# Hajj Readiness Power BI Dashboard

## Overview
This project presents a Power BI dashboard designed to monitor Hajj readiness and operational KPIs.
It focuses on contract tracking, accommodation readiness, and pilgrim distribution to support data-driven decision making.

## Objectives
- Monitor the number of arriving pilgrims
- Track contract utilization and remaining room capacity
- Identify critical contracts with low availability
- Support operational readiness during Hajj season

## Key KPIs
- Total number of pilgrims
- Number of active contracts
- Critical contracts (low remaining rooms)
- Pilgrims distribution by date and contract type

## Data & Tools
- Tool: Microsoft Power BI
- Data Source: Operational Excel datasets
- Modeling: Data modeling and relationships
- DAX: Custom measures for KPIs and conditional logic
